We provide simple scripts to demonstrate the basic functionality of the package.
For more advanced usage or example's source code, please refer to the `examples/` folder in the repository.

.. contents:: Table of Contents:
   :local:
   :depth: 2

-----------
Quick Start
-----------

Package provides functionality to calculate Gower's distance or similarity between two data points.
It supports different types of features, as follows:

-   ratio_scale_interval
-   numeric
-   categorical_nominal
-   categorical_ordinal
-   binary_symmetric
-   binary_asymmetric

.. note::

   Any possible mistakes in feature type specification may lead to incorrect results.
   Their more detailed description can be found in the main repository section.
