What is more, if you would like to contribute, please refer to the Contribution section
mentioned below.

.. contents:: Table of Contents:
   :local:
   :depth: 2

--------------
How to install
--------------

You can install the gower_metric package via pip:

.. code-block:: console

   pip install gower-metric

or add package to your project dependencies using uv:

.. code-block:: console

   uv add gower-metric
